版本控制管理工具 Git 1. Git配置：权限 -> 生成密钥 -> 托管公钥 -> 全局配置 1. 基本操作： 1. 获取仓库权限： 2. 下载Git； 3. 电脑上生成ssh密钥，打开终端，执行 ssh-keyken -t rsa -C “邮箱地址” 4. 执行成功后，切换至 ~/.ssh目录下， 复制id_rsa.pub的内容； 5. 以Github为例，进入 settings -> SSH and GPG keys ，通过cat命令查看id_rsa.pub的内容，然后复制过来，点击add ssh key，相当于将公钥放到了Github上托管 6. 全局配置： git config --global user.name "xxx" git config --global user.email "xxx@xx.com" git config --local user.name "liuxiner" git config --local user.email "1223495869@qq.com" git config --global user.name "bella123987" git config --global user.email "bella@vibe.us" 验证github配置是否正确：
